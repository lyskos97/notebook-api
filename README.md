# Notebook API

Simple Node.JS API with minimalistic CRUD functionality.

## Includes

- Express RESTful API
- Express GraphQL API

## Featuring

- MongoDB/Mongoose
- Typescript
- JWT for authentication
- GraphQL Tools

## Running

First of all set the environment variable `MONGO_URI` to point to your DB.

```bash
npm run dev

# or

yarn dev
```
